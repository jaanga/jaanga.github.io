The code used to display this document is sourced from the code used to display the documentation for Three.js - a JavaScript library for 3D visualizations such as the demo included in this document.

The code - which appears not to have a name - enables a multi-page document to be generated from files stored on GitHub.

  • Allows the two frames to scroll independently.
  • Allows pages to be loaded without reloading the entire frame
  • Overcomes the inability of the GitHub server to update IFRAMES using HTML by providing a JavaScript routine to do this.
  • Provides a simple folder and file structure for creating documents of any length or complexity
  • Provides a simple wiki-like method for creating internal and external links 

A GitHub 'organization' is being used to host the files because:

  • Allows multiple people to 'own' and edit the content and code.
  • Provides robust version control
  • It's free.

Editing large amounts of text in HTML using a plain text editor quickly becomes not fun, There are a number of WYSIWYG editors out there. A current and interesting tool is available at shiftedit.net and is the online-only IDE being used to write this text.

The font used in this document is Inconsolata sourced from Google Web Fonts.
